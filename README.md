# media-player-csharp
C# ile yapmış olduğum media player | Seçilen medya dosyalarını bilgisayarınıza bağlı olan ekranlar arasında kullanıcı tercihine göre oynatır. Videolar kullanıcı tercihine bağlı olarak tekrar oynatılabilir veya tek sefer oynatılabilir. Ekran çözünürlükleri kullanıcı tercihine göre değiştirilebilir.
